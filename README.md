## Run application

In the project directory (in /server and /client dir), you can run in console:

### `yarn i`

or if you use npm, delete yarn.lock and run this:

### `npm i`

Or

Install with installing script:

### `start install.bat`

For start dev server:

### `yarn dev`

or

### `npm run dev`
